Existem técnicas

Inline

A que usei no titulo H1 é a inline, ela é uma tecnica que é melhor evitar. Se for usar apenas para configurções muito pontuais (mudei a fonte)

Internal

Ela fica dentro da tag head e tbm use apenas para configurações pequenas (as H2s estão no formato internal)

External

Ela se deve usar sempre, com ela o arquivo html ficara mais organizado. E tbm fica mais facil de reciclar para colocar em outras páginas (a cor de fundo,o formato e cor das letras estão no formato external)
