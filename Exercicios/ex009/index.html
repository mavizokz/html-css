Listas OL e UL

Lista Ol

Serve para as ordered lists, todas aquelas listas onde a ordem dos itens importa, para delimitar a lista usamos <li>

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Lista UL

As lista unordered lists, todas aquelas lista onde a ordem não importam e para delimitar a lista usamos <li>

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Misturando as coisas/Exercício

Misturando as listas

Podemos colocar listas dentro de listas:

  1. Item 1
  2. Item 2
    • Item 2.1
    • Item 2.2
    • Item 2.3
  3. Item 3
  4. Item 4

Lista de Definição

É uma lista sem demarcadores, como se fosse um dicionario que temos os termos e as suas descrições. Toda lista de Definição está dentro da tag <dl>, cada termo é um <dt>(definition terma) e cada descrição é um <dd> (definition descripion)

HTML
Linguagem de marcação utilizada para criar conteúdo de sites.
CSS
Liguagem de marcação para a especificação de estilos de sites.
JavaScript
Linguagem de programação para criar interatividades em sites.
